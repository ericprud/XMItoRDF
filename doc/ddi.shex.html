This is the ShExC schema for DDI-4.

This model is pretty stable with the exception of:

  • missing shapes: ddi:CatalogItem, ddi:AnalyticMetadatum, ddi:CommonDataElement, ddi:DataCollection, ddi:LogicalResource, ddi:LogicalSegment, ddi:PhysicalSegment, ddi:ProcessSequence, ddi:Signifier
